# Amazon Web Services (AWS)

**Type:** technology-platform

**Description:** Cloud-computing division of Amazon providing infrastructure, analytics, and machine-learning services used by CPG vendors to automate forecasting.

**Key facts:**
- Offers [[entities/amazon-forecast.md|Amazon Forecast]], a fully managed time-series forecasting service requiring no deep ML expertise.
- Provides reference architectures that automate ingestion of Amazon Vendor Central PO data via APIs.
- Supports probabilistic forecasting and SKU-class matching (Class A/B/C) to improve inventory positioning.
