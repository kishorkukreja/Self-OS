# Amazon Forecast

**Type:** tool

**Description:** Fully managed cloud service from AWS that applies Amazon’s internal forecasting expertise to customer data.

**Key facts:**
- Evolved from Amazon’s decade-long internal forecasting journey through statistical, ML, and deep-learning phases.
- Generates probabilistic forecasts rather than single point estimates.
- Can be matched to Class A, B, and C SKU velocity tiers for differentiated inventory policies.
- Accessible via AWS console and APIs; does not require deep time-series expertise.
