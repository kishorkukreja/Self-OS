# Wrap-Up Skill Discovery

**Category:** workflow

**Description:** End-of-session wrap-up can convert implementation experience into durable project memory, reusable patterns, and skill updates.

## The Pattern

Capture what was built, decisions made, reusable procedures discovered, and unresolved follow-up work before the coding context disappears. Use raw session artifacts as the traceable source of truth and promote only stable, reusable practices into wiki patterns or agent skills.

## When to use

Use this after coding sessions that changed workflows, debugging methods, repo conventions, or agent procedures.

## Implementation

1. Read the session summary and decisions.
2. Extract project changes, durable decisions, and reusable procedures.
3. Update project memory and create a pattern only when the lesson is reusable beyond one file.
