# Daily Operating Brief

**Category:** workflow

**Description:** A recurring project-health artifact that summarizes system state and suggests the next move.

## The Pattern
Generate a concise operating brief from repository status, recent commits, PRs, cron health, and pending operational risks. Keep the raw brief as the source of truth and compile only safe summary blocks into the project wiki.

## When to use
Use for long-running repos where scheduled jobs, branches, and knowledge-base updates need daily review.
