# Browser AI APIs

**Definition:** Browser AI APIs expose local or bundled AI models to web applications through browser-level interfaces.

**Why it matters:** The newsletter digest notes controversy around Chrome's Prompt API and local Gemini Nano access. The topic matters because browser vendors can turn model availability, user permission, privacy, and standards governance into platform-level questions.
