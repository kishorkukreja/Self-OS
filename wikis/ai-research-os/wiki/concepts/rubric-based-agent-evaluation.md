# Rubric-Based Agent Evaluation

**Definition:** Rubric-based agent evaluation scores agent outputs or trajectories against explicit criteria for quality, safety, completeness, and task success.

**Why it matters:** The Claude Code video describes outcomes and managed agents that can iterate against a rubric. This is central to long-running agents because it gives automation a checkable definition of done rather than relying on a single model response.
