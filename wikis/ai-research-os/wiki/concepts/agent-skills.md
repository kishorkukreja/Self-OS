# Agent Skills

**Definition:** Agent Skills is a recurring topic captured in this wiki source set.

**Why it matters:** ibelick/ui-skills is the GitHub repository behind UI Skills, a public collection and website for “Skills for Design Engineers.” The project is built primarily with TypeScript and Astro and publishes the UI Skills website, catalog, installer, and skill content. The repository is MIT licensed and, at extraction time, had about 1.5k stars and 66 forks.
