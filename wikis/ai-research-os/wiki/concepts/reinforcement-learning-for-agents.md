# Reinforcement Learning for Agents

**Definition:** Reinforcement learning for agents uses scored task rollouts and environment feedback to improve the model or adapter that drives an agent.

**Why it matters:** Agent work often has observable outcomes: tests pass, a browser task succeeds, a customer-support simulation resolves, or a rubric gives a trajectory score. Platforms such as [[entities/prime-intellect]] Lab package these outcomes into environments so teams can evaluate, train, inspect, deploy, and repeat.
