# Managed Agents

**Definition:** Managed agents are agent systems where the platform owns scheduling, memory, orchestration, and result delivery rather than exposing only a one-shot chat turn.

**Why it matters:** Claude Managed Agents adds dreaming, multiagent orchestration, outcome grading, and webhooks, indicating that agent products are being packaged as managed work queues. This belongs beside [[concepts/subagent-orchestration]] and [[concepts/agentic-memory]] because it shifts agents from prompt sessions to lifecycle-managed automation.
