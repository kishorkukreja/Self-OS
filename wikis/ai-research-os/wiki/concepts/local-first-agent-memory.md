# Local-First Agent Memory

**Definition:** Local-first agent memory keeps durable context in user-controlled storage instead of only in hosted services or opaque model state.

**Why it matters:** Rowboat stores work memory as an Obsidian-compatible Markdown vault. This supports inspection, editing, backup, deletion, and integration with existing note workflows. For Self-OS, the same principle appears in raw captures and compiled wiki pages: durable context should remain user-owned and legible.
