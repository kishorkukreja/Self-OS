# Multi-Agent Orchestration

**Definition:** Multi-agent orchestration coordinates multiple specialized agents so they can divide work, share state, use tools safely, and converge on a goal.

**Why it matters:** The Claude Code platform video frames managed agents as a first-class product surface with roles, hierarchy, workspace sharing, and coordination patterns. This moves multi-agent workflows from ad hoc prompting toward explicit runtime design.
