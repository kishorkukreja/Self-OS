# Skill Evolution

**Skill evolution** refers to the automatic improvement of agent capabilities through usage rather than manual updates. In systems like [[SkillClaw]], trajectories from multiple users are aggregated, analyzed by an agentic evolver, and converted into skill refinements or new skill creation. This concept distinguishes itself from static skill libraries by enabling cross-user knowledge transfer and continuous capability improvement without manual curation.

Key properties include:
- **Collective evolution:** Improvements discovered by one user propagate to all users.
- **Automatic updates:** No explicit human intervention is required.
- **Agentic reasoning:** Updates are generated through open-ended LLM reasoning over evidence, not fixed rules.

Related systems: [[SkillClaw]], [[OpenClaw]], [[Graph of Skills]].
