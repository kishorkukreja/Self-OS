# Agent Benchmarks

**Definition:** Agent benchmarks evaluate whether a tool-using model can complete realistic multi-step tasks, not just answer static questions.

**Why it matters:** The 2026-05-08 X/Twitter digest groups legal-agent, CRM-agent, lifelong-learning, and combinatorial-optimization benchmarks. These sources show the field moving toward domain-specific tasks, trajectory scoring, and long-horizon reliability measures.
