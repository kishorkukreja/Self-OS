# Tree-indexed Document Memory

**Definition:** Tree-indexed document memory stores or derives a hierarchical representation of documents and lets agents navigate that tree before reading evidence.

**Why it matters:** PageIndex File System extends the single-document tree idea to file-level corpora, making the pattern relevant to Self-OS style knowledge bases. It is especially useful when citations and section-level traceability matter more than fuzzy semantic matching.
