# Knowledge Graph Memory

**Definition:** Knowledge graph memory represents context as linked entities and relationships rather than isolated notes or undifferentiated chunks.

**Why it matters:** Rowboat turns email, meetings, notes, and voice memos into a long-lived graph of people, projects, decisions, commitments, and topics. This lets the agent prepare briefs, draft follow-ups, and maintain live notes from accumulated context.
