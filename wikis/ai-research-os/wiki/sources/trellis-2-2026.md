# Microsoft TRELLIS.2: Native and Compact Structured Latents for 3D Generation

**Type:** repo  
**Date:** 2026-04-29  
**URL:** https://github.com/microsoft/TRELLIS.2  
**Raw file:** [[../raw/repos/trellis-2-2026-04-29.md]]

**Summary:** microsoft/TRELLIS.2 is Microsoft's repository for Native and Compact Structured Latents for 3D Generation. It is a 3D generation research/codebase release with assets, configs, data tooling, an o-voxel component, and the trellis2 implementation. The repository has released training code and appears to be a major continuation of Microsoft's TRELLIS line of 3D generative models.
The main reason to track it is that 3D generation is moving from demo-only image-to-3D outputs toward reusable structured latent representations, training pipelines, and compact scene/object generation systems. TRELLIS.2 likely matters for AI tooling that needs 3D asset generation, simulation data, embodied AI environments, or creative pipelines.

**Key contributions:**
- Public Microsoft research repository for compact structured latents in 3D generation.
- GitHub page reports ~6.4k stars and hundreds of forks at ingest time.
- Repo includes assets/, configs/, datatoolkit/, o-voxel/, and trellis2/ directories.
- Latest visible GitHub commit in extraction: Release Training Code by JeffreyXiang, Jan 10 2026.
- Useful for tracking open-source 3D generation tooling and model-training pipelines.
