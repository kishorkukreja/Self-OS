Agent of Empires (AoE): AI Agent Session Manager Running one AI agent is easy. Running five of them across different branches, keeping track of which is stuck, which is waiting on input, and which just made a mess of your working tree, becomes a part-time job. AoE makes it a glance: one dashboard, one status column, git worktrees and Docker sandboxes set up for you, and sessions that outlive your terminal. A session manager for AI coding agents on Linux and macOS. Run multiple agents in parallel across different branches, each in its own isolated session.
