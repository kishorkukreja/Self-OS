# Karpathy on LLM Knowledge Bases (April 2026)

**Type:** article
**Date:** 2026-04-02
**URL:** https://x.com/karpathy/status/1907193941248381031
**Raw file:** [[../../raw/articles/2026-04-12-karpathy-llm-os-tweet.md]]

**Summary:** Karpathy described using LLMs to maintain personal knowledge bases as a core workflow — accumulating raw inputs in a `raw/` folder and having the LLM write and maintain a structured `wiki/`. The LLM acts as the operating layer: synthesising, indexing, and querying knowledge rather than just answering one-off questions.

**Key contributions:**
- Named the `raw/` → `wiki/` folder pattern as a practical LLM workflow
- Framed the LLM as an "OS" layer for personal knowledge, not just a search tool
- Identified contradiction detection at ingest time as a key quality mechanism
- Showed this scales: humans drop content into raw/, the LLM handles all synthesis
