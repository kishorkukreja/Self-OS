# Mem0 — The Memory Layer for Personalized AI

**Type:** repo
**Date:** 2026-04-13
**URL:** https://github.com/mem0ai/mem0
**Raw file:** [[../../raw/repos/mem0-2026-04-13.md]]

**Summary:** Mem0 ("mem-zero") is a production-oriented memory layer that enhances AI assistants and agents with intelligent long-term memory. It supports multi-level memory spanning User, Session, and Agent state with adaptive personalisation. The system is available as a fully managed hosted platform or self-hosted open source, with SDKs for Python and Node.js, a CLI, a browser extension, and integrations with LangGraph and CrewAI. Research claims include +26% accuracy over OpenAI Memory on the LOCOMO benchmark, 91% faster responses than full-context approaches, and 90% lower token usage. Mem0 is a Y Combinator S24 company.

**Key contributions:**
- Multi-level memory architecture (User, Session, Agent state)
- Cross-platform SDKs (Python, Node.js, CLI, browser extension)
- Managed cloud service and self-hosted open-source options
- LangGraph and CrewAI integrations for agent workflows
- Published research on scalable long-term memory for production agents
