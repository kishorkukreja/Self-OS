# Matt Pocock's Agent Skills

**Type:** repo
**Date:** 2026-04-26
**URL:** https://github.com/mattpocock/skills
**Raw file:** [[../../raw/repos/mattpocock-skills-2026-04-26.md]]

**Summary:** This repository contains Matt Pocock's personal agent skills for "real engineering, not vibe coding." The 15+ skills are organised into planning and design (to-prd, to-issues, grill-me, design-an-interface, request-refactor-plan), development (tdd, triage-issue, improve-codebase-architecture), tooling (setup-pre-commit, git-guardrails-claude-code), and writing/knowledge work (write-a-skill, edit-article, ubiquitous-language, obsidian-vault). Each skill is designed as a markdown file with reference data for progressive disclosure, installable via the `skills` CLI using `npx skills@latest add`.

**Key contributions:**
- Structured skill taxonomy covering the full engineering lifecycle
- Progressive disclosure pattern: skill file defines *what to do*, ref file provides *what to know*
- TDD skill with explicit red-green-refactor loop and vertical slicing
- Ubiquitous-language skill for extracting DDD-style glossaries from conversations
- Obsidian-vault skill for structured note management with wikilinks
