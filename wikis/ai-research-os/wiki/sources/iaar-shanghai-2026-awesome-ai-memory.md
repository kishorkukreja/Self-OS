# Awesome-AI-Memory Survey Repository

**Type:** repo
**Date:** 2026-04-13
**URL:** https://github.com/IAAR-Shanghai/Awesome-AI-Memory
**Raw file:** [[../../raw/repos/awesome-ai-memory-2026-04-13.md]]

**Summary:** Awesome-AI-Memory is a systematically curated knowledge base that maps the rapidly evolving research landscape of LLM memory systems. It bridges natural language processing, information retrieval, intelligent agent systems, and cognitive science by cataloguing over 300 research papers and 90 open-source projects. The repository provides a unified taxonomy that classifies memory along multiple dimensions: storage location (parametric vs external), temporal scope (short-term vs long-term), content type (episodic, semantic, procedural), and operational mechanics (writing, retrieval, updating, forgetting, compression).

**Key contributions:**
- Multi-dimensional memory taxonomy spanning storage, time, content, and architecture
- Comprehensive coverage of memory operations: writing, retrieval, updating, deletion, compression
- Curated benchmarks and evaluation datasets for LLM memory systems
- Classification of memory in agent systems: single-agent, multi-agent shared, tool-augmented, planning-aware
- Distinguishes agent memory from related concepts like RAG and context engineering
