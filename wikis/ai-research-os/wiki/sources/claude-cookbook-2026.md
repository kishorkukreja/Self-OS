# Claude Cookbook — Official Anthropic Practical Guides

**Type:** article
**Date:** 2026-04-12
**URL:** https://platform.claude.com/cookbook/
**Raw file:** [[../../raw/resources/Claude Cookbook.md]]

**Summary:** Anthropic's official library of practical guides and code examples for building with Claude. Covers agent patterns, tool use, RAG & retrieval, multimodal, managed agents, Claude Agent SDK, and skills. Updated regularly — as of April 2026 the latest additions focus on Managed Agents (sandboxed environments, session management, prompt versioning) and context engineering.

**Key contributions:**
- Managed Agents tutorials: sandboxed agent environments, file mounts, session lifecycle, webhook patterns
- Context engineering: memory compaction, tool clearing, automatic context compaction
- Programmatic tool calling (PTC): agents write code to call tools, reducing latency and token use
- Tool search with embeddings: scale to thousands of tools via semantic discovery
- Claude Agent SDK cookbook series: chief-of-staff, observability, SRE incident responder, session browser
- Knowledge graph construction with Claude: entity extraction, relation mining, multi-hop querying
