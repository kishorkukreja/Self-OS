# Awesome Agent Harness — Implementation-First Harness Engineering Catalog

**Type:** repo

**Raw file:** [[../raw/repos/awesome-agent-harness-2026-05-04.md]]

## Summary
Awesome Agent Harness is a catalog rather than a single implementation, but it is valuable because it maps the harness-engineering landscape across architecture, context/state engineering, sandboxing, protocols, evaluation, observability, governance, and reference systems. For Self-OS, this source can seed future research requests and prevent tunnel vision around one framework. It also confirms that harness engineering is becoming a recognizable implementation discipline: practitioners are collecting patterns for state, permissions, benchmarks, sandboxes, observability, and safety as separate concerns that surround the model. As a meta-source, it should be revisited when new Self-OS research asks for alternatives, benchmarks, or implementation references in the agent-harness ecosystem.

## Key contributions
- Catalogs more than 160 implementation-focused harness resources.
- Organizes the field into architecture, context, sandboxing, protocols, evaluation, observability, and governance.
- Serves as a discovery map for future Self-OS agent-infrastructure research.
