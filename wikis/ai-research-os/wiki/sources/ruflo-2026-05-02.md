## Summary

Ruflo is a comprehensive multi agent AI orchestration framework designed specifically for Claude Code. It transforms Claude Code into a development platform with 100+ specialized agents that can self organize into swarms, learn from past tasks, and retain memory across sessions. The project ships as a Claude Code native plugin with 32 plugin modules covering orchestration, memory, code quality, security, and federation. Originally called Claude Flow, Ruflo is authored by ruvnet and uses WASM kernels written in Rust to power its policy engine, embeddings, and proof system. It supports plugin installation through Claude Code commands and also has a web UI and goal planner. Ruflo represents one of the most ambitious attempts to build an agent operating system on top of Claude Code. For SelfOS style agent workflows, the plugins architecture (swarm, autopilot, federation, hooks) and the RuVector memory/intelligence layer are especially relevant. It is a strong candidate for studying agent orchestration patterns, but should be vetted for real world reliability before depending on it in production. Claude Code plugin : installed via /plugin install ruflo core@ruflo (and companion plugins for swarm, autopilot, federation, etc.) 32 plugins organized into categories: Core & Orchestration, Memory & Knowledge, Intelligence & Learning, Code Quality & Testing, Security & Compliance, Arc, and more. Self learning architecture : agents learn from successful patterns and improve over time via a closed learning loop (RETRIEVE → JUDGE → DISTILL → CONSOLIDATE → ROUTE). RuVector intelligence : includes components like SONA (self optimizing pattern learning), EWC++ (anti forgetting), Flash Attention, HNSW, ReasoningBank, Hyperbolic embeddings, LoRA/MicroLoRA, Int8 quantization, SemanticRouter, and 9 RL algorithms. Swarm coordination : supports mesh, hierarchical, ring, and star topologies with consensus mechanisms (Raft / BFT / Gossip). Federation : agents across different machines can collaborate securely. Hooks system : automatically routes tasks in the background after init. MCP integration : exposes tools via MCP server. Web UI : flo.ruv.io and goal planner at goal.ruv.io.
