# notebooklm-py — Unofficial Python API and CLI for Google NotebookLM

**Type:** repo
**Date:** 2026-04-12
**URL:** https://github.com/teng-lin/notebooklm-py
**Raw file:** [[../../raw/repos/notebooklm-py-repo.md]]

**Summary:** Unofficial Python library and CLI providing full programmatic access to Google NotebookLM, including capabilities not exposed in the web UI. Ships with a native Claude Code skill integration and Codex AGENTS.md. Key use case for Self OS: automate deep research pipelines from Claude Code and VPS agents.

**Key contributions:**
- Python API + CLI for all NotebookLM operations (notebooks, sources, chat, research, artifacts)
- Claude Code skill integration: `notebooklm skill install` for local use
- Agent integration: works with Claude Code, Codex, OpenClaw
- Research automation: bulk-import sources, run web/Drive research queries programmatically
- Content generation: audio overviews, videos, slide decks, quizzes, flashcards, reports, mind maps
- Features beyond web UI: batch downloads, structured quiz/flashcard export, mind map JSON

**Caution:** Uses undocumented Google APIs — not affiliated with Google, may break on API changes.
