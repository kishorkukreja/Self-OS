## Summary

Refero Styles is a curated design reference search tool for extracting and browsing design taste from real brands and products. It lets users search references by brand, mood, color, typography, or URL, then inspect styles in terms of colors, type, spacing, components, and a DESIGN.md file that an AI agent can use. For Self OS design creation workflows, this is useful as a practical inspiration and prompt/context source when asking coding agents to produce higher taste UI rather than generic AI looking interfaces. Core positioning: “Design taste, extracted.” Status: Beta. Primary use case: Search curated real product references and convert taste into agent usable design context. Search axes: brand, mood, color, typography, and URL. Suggested categories: Minimal Design, Clean SaaS, Editorial Type, Soft Gradients, Monochrome UI, Playful Design, High Contrast, Premium Design. Each style appears to expose: colors, typography, spacing, components, and a DESIGN.md for agents. Agent relevance: complements the design md skill by providing real world style references and DESIGN.md examples that can be passed into coding/design agents. MCP relevance: promotes Refero MCP, which gives AI coding agents searchable access to product screens and user flows.
