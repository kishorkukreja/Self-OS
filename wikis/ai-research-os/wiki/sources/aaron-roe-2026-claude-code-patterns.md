# Claude Code Pattern Library

**Type:** repo
**Date:** 2026-04-13
**URL:** https://github.com/AaronRoeF/claude-code-patterns
**Raw file:** [[../../raw/repos/claude-code-patterns-2026-04-13.md]]

**Summary:** This community-maintained pattern library compiles 137 real-world techniques for getting the most out of Claude Code. It is organised into five parts: patterns worth sharing (Two OS architecture, trigger-phrase skills, MCP-first data gathering), 97 pro-tips across 15 categories, the AI Wiki Pattern for building persistent knowledge bases with Claude Code and Obsidian, quick reference cheat sheets, and live implemented examples. The document emphasises practical operational leverage: global MCP configuration, confirmation gates on destructive actions, pre-approved permissions for speed, structured output locations, environment-aware skills, user identity systems, and hook-driven self-maintenance.

**Key contributions:**
- Two OS architecture (WorkOS vs PersonalOS) with separate CLAUDE.md files
- Trigger-phrase skill architecture using natural language intent matching
- MCP-first data gathering pattern for rich, automated briefings
- Hook-driven self-maintenance: auto-updating tests, MEMORY.md logs, and remote approval
- AI Wiki Pattern for compounding knowledge bases inspired by Karpathy
