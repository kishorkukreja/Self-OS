# Hermes Agent

**Type:** tool

**Description:** Open-source AI agent framework by Nous Research that learns from experience by saving reusable procedures as skills and persists memory across sessions via pluggable backends.

**Key facts:**
- Provider-agnostic: works with OpenRouter, Anthropic, OpenAI, DeepSeek, local models, and 15+ others
- **Skills system**: solves complex problems once, then saves the workflow as a reusable `SKILL.md` in `~/.hermes/skills/`
- Skills accumulate over time, making the agent better at specific tasks and environments
- **Persistent memory**: built-in, Honcho, Mem0, and other pluggable backends
- Multi-platform gateway: runs on Telegram, Discord, Slack, WhatsApp, Signal, Matrix, Email, and 10+ other platforms
- Supports cron jobs, webhooks, MCP servers, subagent delegation, and voice (STT/TTS)
- **Profiles**: run multiple isolated instances with separate configs, sessions, skills, and memory
- Can spawn additional Hermes instances as fully independent subprocesses for parallel work
- Source: https://github.com/NousResearch/hermes-agent
