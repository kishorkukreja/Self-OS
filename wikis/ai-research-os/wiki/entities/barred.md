# BARRED

**Type:** tool

**Description:** BARRED — Boundary Alignment Refinement through REflection and Debate — is a framework for training custom policy guardrail classifiers from only a task/policy description and a small set of unlabeled examples. The core claim is that custom guardrail deployment is blocked less by model architecture and more by the cost and difficulty of labeled data
