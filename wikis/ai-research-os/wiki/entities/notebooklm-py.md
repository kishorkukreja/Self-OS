# notebooklm-py

**Type:** tool

**Description:** Unofficial Python library and CLI that provides full programmatic access to Google NotebookLM — used in Self OS for deep research automation from Claude Code and VPS agents.

**Key facts:**
- Created by teng-lin (GitHub: teng-lin/notebooklm-py)
- Ships with a native Claude Code skill: `notebooklm skill install`
- CLI command: `notebooklm` (login, list, create, ask, source add, generate, download)
- Uses undocumented Google APIs — unofficial, may break on Google changes
- Installed in Self OS at: `C:/Users/Kish Kukreja/AppData/Local/Programs/Python/Python312/Scripts/notebooklm.exe`
- Auth storage: `C:/Users/Kish Kukreja/.notebooklm/storage_state.json`
- Used in Self OS via `/notebooklm-research` Claude Code command
