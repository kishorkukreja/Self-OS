# OpenClaw

**Type:** tool

**Description:** Open-source personal AI assistant that runs locally and autonomously, capable of self-improving by writing its own skills and maintaining persistent memory via markdown memory files.

**Key facts:**
- Created by Peter Steinberger (@steipete); mascot is a lobster 🦞
- Runs on macOS, Linux, and Windows via one-liner install (`curl -fsSL https://openclaw.ai/install.sh | bash`)
- Local-first: context and skills live on the user’s machine, not a walled garden
- Can write its own skills and hot-reload them without restart
- Persistent memory via workspace markdown files: `AGENTS.md`, `SOUL.md`, `IDENTITY.md`, `TOOLS.md`, `USER.md`
- Supports 50+ integrations including Claude, GPT, Spotify, Philips Hue, Obsidian, Twitter/X, Gmail, GitHub
- Community skill registry called **Clawhub**
- Can operate on scheduled cron jobs and a heartbeat checked every 30 minutes
- Featured by NVIDIA as a runtime for local autonomous agents (NVIDIA NemoClaw)
- Communication via WhatsApp, Telegram, Discord, Slack, Signal, iMessage, or terminal
