# Obsidian

**Type:** tool

**Description:** A local-first markdown note-taking application with graph view, wikilinks, and a rich plugin ecosystem — the recommended frontend for LLM knowledge bases.

**Key facts:**
- Works with plain markdown files in a local folder (vault)
- Graph view visualises wikilink connections between notes
- Obsidian Web Clipper extension saves web pages directly to vault as markdown
- Free tier covers all core features; sync is paid
- Works on Mac, Windows, Linux, iOS, Android
- Used in the Self OS setup as the visual IDE for the knowledge-base repo
