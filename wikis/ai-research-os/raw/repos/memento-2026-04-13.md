     9|# Memento
    10|
    11|[**Paper (PDF)**](docs/memento.pdf) | [**OpenMementos Dataset**](https://huggingface.co/datasets/microsoft/OpenMementos)
    12|
    13|**Memento** extends the effective output length of large language models by splitting chain-of-thought reasoning into **blocks** and **summaries** (memento). After each reasoning block, the model generates a short summary, then the block content is evicted from the KV cache. The model continues from the summary with a shorter context, enabling more reasoning within a fixed context window.
    14|
    15|## Special tokens
    16|
    17|| Token | Purpose |
    18||-------|---------|
    19|| `<think>` / `</think>` | Reasoning wrapper |
    20|| `<\|block_start\|>` / `<\|block_end\|>` | Reasoning block boundaries |
    21|| `<\|summary_start\|>` / `<\|summary_end\|>` | Summary (memento block) boundaries |
    22|
    23|**Block structure:** `<|block_start|> reasoning <|block_end|> <|summary_start|> summary <|summary_end|>`
    24|
    25|## Repository layout
    26|

    32|## Quick start
    33|
    34|### Data pipeline
    35|
    36|Convert chain-of-thought traces into Memento training data:
    37|
    38|```bash
    39|pip install -r data/requirements.txt
    40|export OPENAI_API_KEY=***   # or any OpenAI-compatible provider
    41|
    42|cd data/pipeline
    43|python run_full_pipeline.py \
    44|    --input ../examples/example_trace.jsonl \
    45|    --output-dir output/ \
    46|    --model gpt-4o \
    47|    --limit 1
    48|```
    49|
    50|See [data/README.md](data/README.md) for full documentation.
    51|
    52|### vLLM inference with block masking
    53|
    54|* Step 1: Set Up the Environment
    55|    Build a customized vllm with block masking support:
    56|    ```bash
    57|    pip install vllm==0.13.0
    58|    cd vllm
    59|    bash install_overlay.sh
    60|    ```
    61|
    62|* Step 2: Serve a Memento Model with KV Cache Compaction
    63|    To expose the model through an API-compatible server, run:
    64|    ```bash
    65|    python -m vllm.entrypoints.openai.api_server \
    66|        --model /path/to/memento-checkpoint \
    67|        --served-model-name memento \
    68|        --port 8010 \
    69|        --max-model-len 32768 \
    70|        --gpu-memory-utilization 0.9 \
    71|        --trust-remote-code \
    72|        --chat-template chat_templates/memento_nosys.jinja \
    73|        --block-masking-config '{
    74|            "enable": true,
    75|            "keep_last_n_blocks": 0,
    76|            "mask_delimiters": false,
    77|            "compact_on_summary_end": true,
    78|            "require_assistant_section": true,
    79|            "debug": true
    80|        }'
    81|    ```
    82|
    83|    See [vllm/README.md](vllm/README.md) for full documentation, including API usage and alternative setup options.
