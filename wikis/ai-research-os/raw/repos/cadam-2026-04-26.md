    28|## What It Is
    29|
    30|> **"The Open Source Text to CAD Web App"**
    31|
    32|CADAM transforms natural language descriptions and images into parametric 3D models directly in the browser. It generates OpenSCAD code and renders it via WebAssembly, allowing real-time parameter adjustments without re-invoking AI generation.
    33|
    34|---
    35|
    36|## Core Features
    37|
    38|- **AI-Powered Generation** — Natural language and image-to-3D model generation
    39|- **Parametric Controls** — Interactive sliders for instant dimension adjustments
    40|- **Multiple Export Formats** — `.STL` and `.SCAD`
    41|- **Browser-Based** — Runs entirely client-side using WebAssembly
    42|- **Library Support** — Includes BOSL, BOSL2, and MCAD libraries
    43|- **Custom Fonts** — Built-in Geist font support for text in models
    44|
    45|### Key Capabilities
    46|
    47|| Feature | Description |
    48||---------|-------------|
    49|| Natural Language Input | Describe models in plain English |
    50|| Image References | Upload images to guide generation |
    51|| Real-time Preview | Three.js rendering |
    52|| Parameter Extraction | Auto-identifies adjustable dimensions |
    53|| Smart Updates | Efficient parameter changes without AI re-generation |
    54|

    56|
    57|## Tech Stack
    58|
    59|- **Frontend:** React 18 + TypeScript + Vite
    60|- **3D Rendering:** Three.js + React Three Fiber
    61|- **CAD Engine:** OpenSCAD WebAssembly
    62|- **Backend:** Supabase (PostgreSQL + Edge Functions)
    63|- **AI:** Anthropic Claude API (via OpenRouter)
    64|- **Styling:** Tailwind CSS + shadcn/ui

    68|## Quick Start
    69|
    70|```bash
    71|git clone https://github.com/Adam-CAD/CADAM.git
    72|cd CADAM
    73|npm install
    74|npx supabase start
    75|npx supabase functions serve --no-verify-jwt
    76|npm run dev
