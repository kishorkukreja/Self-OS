## What It Is

> **"The Open Source Text to CAD Web App"**

CADAM transforms natural language descriptions and images into parametric 3D models directly in the browser. It generates OpenSCAD code and renders it via WebAssembly, allowing real-time parameter adjustments without re-invoking AI generation.

---

## Core Features

- **AI-Powered Generation** — Natural language and image-to-3D model generation
- **Parametric Controls** — Interactive sliders for instant dimension adjustments
- **Multiple Export Formats** — `.STL` and `.SCAD`
- **Browser-Based** — Runs entirely client-side using WebAssembly
- **Library Support** — Includes BOSL, BOSL2, and MCAD libraries
- **Custom Fonts** — Built-in Geist font support for text in models

### Key Capabilities

| Feature | Description |
|---------|-------------|
| Natural Language Input | Describe models in plain English |
| Image References | Upload images to guide generation |
| Real-time Preview | Three.js rendering |
| Parameter Extraction | Auto-identifies adjustable dimensions |
| Smart Updates | Efficient parameter changes without AI re-generation |

---

## Tech Stack

- **Frontend:** React 18 + TypeScript + Vite
- **3D Rendering:** Three.js + React Three Fiber
- **CAD Engine:** OpenSCAD WebAssembly
- **Backend:** Supabase (PostgreSQL + Edge Functions)
- **AI:** Anthropic Claude API (via OpenRouter)
- **Styling:** Tailwind CSS + shadcn/ui

---

## Quick Start

```bash
git clone https://github.com/Adam-CAD/CADAM.git
cd CADAM
npm install
npx supabase start
npx supabase functions serve --no-verify-jwt
npm run dev
```
