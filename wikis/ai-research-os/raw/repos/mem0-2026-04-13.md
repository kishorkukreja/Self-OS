    64|# Introduction
    65|
    66|[Mem0](https://mem0.ai) ("mem-zero") enhances AI assistants and agents with an intelligent memory layer, enabling personalized AI interactions. It remembers user preferences, adapts to individual needs, and continuously learns over time�ideal for customer support chatbots, AI assistants, and autonomous systems.
    67|
    68|### Key Features & Use Cases
    69|
    70|**Core Capabilities:**
    71|- **Multi-Level Memory**: Seamlessly retains User, Session, and Agent state with adaptive personalization
    72|- **Developer-Friendly**: Intuitive API, cross-platform SDKs, and a fully managed service option
    73|
    74|**Applications:**
    75|- **AI Assistants**: Consistent, context-rich conversations
    76|- **Customer Support**: Recall past tickets and user history for tailored help
    77|- **Healthcare**: Track patient preferences and history for personalized care
    78|- **Productivity & Gaming**: Adaptive workflows and environments based on user behavior

   118|### Basic Usage
   119|
   120|Mem0 requires an LLM to function, with `gpt-4.1-nano-2025-04-14 from OpenAI as the default. However, it supports a variety of LLMs; for details, refer to our [Supported LLMs documentation](https://docs.mem0.ai/components/llms/overview).
   121|
   122|First step is to instantiate the memory:
   123|
   124|```python
   125|from openai import OpenAI
   126|from mem0 import Memory
   127|
   128|openai_client = OpenAI()
   129|memory = Memory()
   130|
   131|def chat_with_memories(message: str, user_id: str = "default_user") -> str:
   132|    # Retrieve relevant memories
   133|    relevant_memories = memory.search(query=message, user_id=user_id, limit=3)
   134|    memories_str = "\n".join(f"- {entry['memory']}" for entry in relevant_memories["results"])
   135|
   136|    # Generate Assistant response
   137|    system_prompt = f"You are a helpful AI. Answer the question based on query and memories.\nUser Memories:\n{memories_str}"
   138|    messages = [{"role": "system", "content": system_prompt}, {"role": "user", "content": message}]
   139|    response = openai_client.chat.completions.create(model="gpt-4.1-nano-2025-04-14", messages=messages)
   140|    assistant_response = response.choices[0].message.content
   141|
   142|    # Create new memories from the conversation
   143|    messages.append({"role": "assistant", "content": assistant_response})
   144|    memory.add(messages, user_id=user_id)
   145|
   146|    return assistant_response
   147|
   148|def main():
   149|    print("Chat with AI (type 'exit' to quit)")
   150|    while True:
   151|        user_input = input("You: ").strip()
   152|        if user_input.lower() == 'exit':
   153|            print("Goodbye!")
   154|            break
   155|        print(f"AI: {chat_with_memories(user_input)}")
   156|
   157|if __name__ == "__main__":
   158|    main()
   159|```
