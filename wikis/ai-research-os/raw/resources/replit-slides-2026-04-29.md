# Replit Slides

## Summary
Replit Slides is an AI slide-deck creation workflow inside Replit Agent. Users describe the deck they want, upload source material or brand guidelines, and Replit generates a complete presentation that can be refined through chat, edited visually, explored in Design Canvas, presented in-browser, deployed as a live URL, or exported to PPTX, Google Slides, and PDF.

The product is notable because Replit treats slide decks as generated software artifacts: the docs say slides are built with React, which is intended to enable pixel-perfect and editable exports rather than static image-only decks. Replit positions the tool for pitch decks, investor presentations, product overviews, operating metrics decks, architecture presentations, and general business storytelling.

## Key points
- Replit Agent can generate complete slide decks from a prompt, document, rough notes, uploaded reference file, or brand guidelines.
- Slide decks support conversational iteration: users can ask to add/remove/reorder slides, change themes, make content more concise, add charts/icons, or adjust layouts.
- Visual editing supports direct changes to text, colors, spacing, and layout without consuming AI credits; drag-and-drop element editing is not yet supported.
- Design Canvas lets users explore alternate design directions for individual slides or deck sections.
- Outputs can be presented in-browser, deployed as shareable URLs, or exported to PPTX, Google Slides, and PDF.
- Replit's marketing page frames the product as a way to turn docs or rough ideas into polished, structured decks with templates, brand import, slide-order customization, and AI-organized narrative flow.

## Why it matters
This is part of a broader shift from chatbots that merely draft slide text to agentic creation tools that produce editable, branded, presentation-ready artifacts. It is especially relevant for AI-assisted content production, internal strategy decks, investor decks, and product/architecture communication.

## Raw content
Replit describes Slide Decks as a way to create polished, professional presentations entirely through conversation with Agent. The user describes the presentation and receives a complete deck with layouts, text, and visuals. After generation, the deck can be refined by chatting with Agent, edited visually, explored through Design Canvas, presented in full-screen mode, deployed as a live URL, or exported.

Two creation paths are documented: type a prompt from the Replit homepage and let Agent detect that the task is a slide deck, or choose Slides from the app-type dropdown and then describe the presentation. Example prompts include creating a pitch deck for an AI-powered fitness app, making a 10-slide investor presentation for a Series A raise, and building a product overview deck for a team offsite.

Replit supports uploaded brand guidelines so generated decks can match colors, fonts, visual style, and broader brand consistency. Users can also upload a reference file or existing presentation/document that Agent uses as a foundation.

Iterative edits can cover content, styling, slides, visuals, and structure. Replit gives examples such as making the problem slide more concise, adding a competitive landscape slide after market size, using a dark theme with blue accents, making fonts larger and more minimal, adding roadmap slides, removing a final slide, adding a projected-growth chart, including feature icons, moving the team slide to the end, or splitting a feature slide into two slides.

The docs emphasize that slides render in a preview pane as changes are made. The Visual Editor allows direct editing of text, colors, spacing, and layout. Replit notes that clicking on an element to adjust it does not consume AI credits. Users can duplicate or delete slides via right-click. Drag-and-drop editing of elements within a slide is not yet supported; users should ask Agent in chat to move or rearrange elements.

Design Canvas allows users to compare alternate visual treatments for slides or sections and bring the preferred version back into the main deck. Presentation mode works from the preview pane, and deployed decks can be shared as browser-viewable links. Mobile rendering is supported.

Replit states that slides are built with React under the hood and can export to fully editable PPTX, Google Slides, or PDF. The product page says users can start with a document, choose a template, and receive polished, structured decks ready to present. It highlights use cases including closing deals, raising a round, sharing architecture, and operating metrics.
