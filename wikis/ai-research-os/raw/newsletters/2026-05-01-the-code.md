## 💡 Insight — AI evals are becoming the new compute bottleneck

- Running a top-tier benchmark now costs as much as a used car per run, per Hugging Face's analysis. ([Source](https://huggingface.co/blog/evaleval/eval-costs-bottleneck))
- Old shortcuts no longer work: agent benchmarks involve long multi-step tasks where cutting corners loses the exact data needed to differentiate models.
- Agents are inconsistent across runs; a Princeton paper shows capability gains haven't improved reliability — leaderboards only report best-case results. ([Source](https://arxiv.org/abs/2602.16666))
- Most teams start evals from scratch because there's no standard format for sharing results, giving only big-budget labs the ability to grade frontier models.
- Hugging Face's Every Eval Ever project is building a shared format to democratize eval. ([Source](https://evalevalai.com/projects/every-eval-ever/))

## 🛠️ AI Coding Hack — How to stop Codex from giving up mid-task

OpenAI's Ralph loop pattern in Codex v0.128.0 lets agents run until your goal is fully finished — including tasks that span days.

Add to `~/.codex/config.toml`:

```toml
[features]
goals = true
```

Then use the `/goal` command:

```
/goal Refactor the auth module to use JWTs, update every call site, and fix any failing tests.
```

You can pause, resume, or clear tasks anytime. Set a token limit in your config first to avoid unexpected costs.

[Source: @thsottiaux](https://x.com/thsottiaux/status/2049970070873629026)
