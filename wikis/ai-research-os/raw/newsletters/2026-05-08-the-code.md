## 💡 Insight — World models are AI's next big bet

- Yann LeCun raised $1.03B for AMI Labs on the world-models premise; Fei-Fei Li and top rivals are chasing the same goal.
- Unlike LLMs predicting the next word, [world models](https://www.nvidia.com/en-in/glossary/world-models/) predict what happens after an action — e.g., pairing dashcam footage with steering inputs to train self-driving systems.
- Data is the primary bottleneck: world models need video paired with the exact actions that generated it. VC James Wang calls this "[data friction](https://www.weightythoughts.com/p/world-models-can-change-everything)."
- Currently viable only in narrow domains (driving, gaming) where action labels are built in.
- World models give AI a practical grasp of physics and consequences that language alone can't replicate. Full case in [LeCun's paper](https://arxiv.org/abs/2603.19312).

## 🛠️ AI Coding Hack — How to run Codex without manual approvals

Codex used to stall every few minutes waiting for permission to run scripts or access the network. OpenAI's head of Codex [shared a fix](https://x.com/thsottiaux/status/2050989326570532919): Auto-review is now the default inside OpenAI, delegating each approval to a separate agent that vets the action against a risk policy. Cuts user prompts by ~200x.

Add this to `~/.codex/config.toml`:

```toml
approvals_reviewer = "auto_review"
```

Restart Codex, and the reviewer takes over. The TUI shows each decision in real time.
