# Hermes Experiment Workspaces

This directory is the canonical Self-OS workspace root for Hermes/agent workflow experiments.

Use one subdirectory per experiment:

```text
hermes-experiments/<slug>/
  README.md
  mission.md
  reward-cycle.md
  artifacts/
  runs/
  reviews/
```

See `/data/Self-OS/docs/hermes-agent-experiment-workspaces.md` for routing rules and verification commands.

Start from `_template/` when creating a new experiment.
